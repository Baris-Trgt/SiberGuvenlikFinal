<br> Uygulamayı test etmek için aşağıdaki değerleri girebilirsiniz. <br/>

<br> Web Sitesi Adresinin Degerini: testphp.vulnweb.com <br/>

<br> Uygulama sonunda JSON fortmatında kullanıcılara çıktı vermektedir.<br/>
